## 专业老师和社区导师,到底选哪个?

italki上有两种老师:专业老师(Professional Teacher)和社区导师(Community Tutor)。

**专业老师**是那种有教学资格证书、多年教学经验的。他们会准备教材,制定学习计划,系统地教你语法、词汇、发音。如果你是零基础,或者准备考雅思、托福这类考试,找专业老师比较靠谱。

**社区导师**就是母语者,没有教师资格证,但他们能陪你练对话,纠正你的发音和表达。如果你已经有一定基础,只是缺乏口语练习,社区导师是很好的选择——而且价格便宜很多。
